# ウェブ制作向けテンプレート集

案件種別ごとの要件定義テンプレートと出力フォーマット

---

## 概要

ウェブ制作の主要な案件種別に対応したテンプレートを提供します。
プロジェクトの種類に応じて適切なテンプレートを選択し、効率的に要件定義を進めてください。

```
→ コーポレートサイトをベースに、ECサイトの要件を追加
```

#### Step 2: 不要な要件を削除

```markdown
例: "採用ページは不要" の場合
→ SR-002（採用応募増加）を削除
→ 関連するSY、SWも削除
```

#### Step 3: 独自要件を追加

```markdown
例: "多言語対応が必要" の場合
→ SR-006: 海外ユーザーに情報を届けたい
→ SY-006: 多言語切り替え機能
→ SW-006: i18n対応（next-i18next）
```

#### Step 4: 品質要件の優先度を調整

```markdown
例: "パフォーマンスよりセキュリティ優先" の場合
→ セキュリティ: 高
→ 性能効率性: 中
```
