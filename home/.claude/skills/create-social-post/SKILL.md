# SNS コンテンツ作成

## Quick Start

1. プラットフォームと目的を確認
2. プラットフォーム別ガイドラインに沿って作成
3. 複数案を提示

## ワークフロー

### Step 1: 要件確認

- [ ] プラットフォーム（Twitter/LinkedIn/Instagram/TikTok）
- [ ] 目的（認知/エンゲージメント/コンバージョン）
- [ ] ターゲット層
- [ ] トーン（カジュアル/プロフェッショナル）

### Step 2: コンテンツ作成

- [ ] プラットフォーム別ガイドラインに従う
- [ ] フック → 本文 → CTA の構成
- [ ] ハッシュタグ選定

### Step 3: 複数案提示

- [ ] 3案以上を作成
- [ ] 各案の狙いを説明

## プラットフォーム別ガイドライン

### Twitter/X

| 項目         | ガイドライン                       |
| ------------ | ---------------------------------- |
| 文字数       | 280文字以内（日本語は140文字目安） |
| ハッシュタグ | 1〜2個に絞る                       |
| 画像         | 1200×675px 推奨                    |

**スレッド構成:**

1. **Hook**: 最初のツイートで興味を引く
2. **Build**: 価値を積み上げる
3. **Climax**: 核心・最大の価値
4. **CTA**: 行動を促す

### LinkedIn

| 項目   | ガイドライン                 |
| ------ | ---------------------------- |
| トーン | プロフェッショナル、B2B向け  |
| 内容   | データ、実績、インサイト重視 |
| 長さ   | 1,300文字以内推奨            |

**構成:**

1. フック（最初の2行で興味を引く）
2. ストーリー/データ
3. 学び/インサイト
4. CTA/質問

### Instagram

| 項目         | ガイドライン                 |
| ------------ | ---------------------------- |
| 重視         | ビジュアルファースト         |
| ハッシュタグ | 5〜15個                      |
| フォーマット | フィード/ストーリーズ/リール |

**キャプション構成:**

1. フック（1行目で興味を引く）
2. 価値提供
3. CTA
4. ハッシュタグ（コメント欄でも可）

### TikTok

| 項目   | ガイドライン                      |
| ------ | --------------------------------- |
| 最重要 | 最初の1〜3秒                      |
| 形式   | トレンドサウンド/フォーマット活用 |
| 内容   | 教育系 or エンタメ系              |

## バイラル要素チェック

- [ ] **感情**: 驚き、共感、喜びを引き出すか
- [ ] **価値**: 役立つ、学びになるか
- [ ] **共有性**: シェアしたくなるか
- [ ] **タイミング**: トレンドや時事と関連があるか

## 出力フォーマット

```markdown
## [プラットフォーム名] 投稿案

### 案1: [コンセプト]
```

[投稿本文]

```text
- 狙い: [この案の意図]
- ハッシュタグ: #tag1 #tag2

### 案2: [コンセプト]
...

### 案3: [コンセプト]
...

## 推奨
[最も効果的と思われる案とその理由]

## 投稿タイミング
- 推奨時間帯: [時間]
- 理由: [根拠]
```
