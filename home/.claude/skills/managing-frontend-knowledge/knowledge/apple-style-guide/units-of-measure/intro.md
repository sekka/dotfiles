# Units of Measure - Introduction

This section provides guidance on using units of measurement consistently in Apple materials.

## Overview

Apple's style guide establishes standards for:
- Prefixes for units of measure
- Names and symbols for various units
- Formatting conventions for measurements

## Key Principles

- Use standard international units (SI units) when appropriate
- Maintain consistency in abbreviations and symbols
- Follow locale-specific conventions for international content
