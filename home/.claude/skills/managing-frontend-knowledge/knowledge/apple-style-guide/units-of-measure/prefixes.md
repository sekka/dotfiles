# Prefixes for Units of Measure

Guidelines for using prefixes with units of measurement in Apple materials.

## Common Prefixes

### SI Prefixes
Standard International System prefixes include:
- **kilo- (k)**: 10³ (thousand)
- **mega- (M)**: 10⁶ (million)
- **giga- (G)**: 10⁹ (billion)
- **tera- (T)**: 10¹² (trillion)

### Binary Prefixes (Computing)
For digital storage and memory:
- **kibi- (Ki)**: 2¹⁰ (1,024)
- **mebi- (Mi)**: 2²⁰ (1,048,576)
- **gibi- (Gi)**: 2³⁰ (1,073,741,824)
- **tebi- (Ti)**: 2⁴⁰ (1,099,511,627,776)

## Usage Guidelines

- Use appropriate prefixes based on context (decimal vs. binary)
- Maintain consistency within documents
- Follow Apple's specific formatting standards
