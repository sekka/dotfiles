# Changes to the Guide

This page documents updates to Apple's official style guide.

## New Entries (Current Version)

The guide added 19 new entries including:
- "Active Noise Cancellation"
- "Apple Intelligence"
- "Apple M4 chip"
- "Genmoji"
- "Image Playground"
- "Writing Tools"

Additional new terms cover audio features, accessibility functions, and emerging product capabilities.

## Changed Entries (Current Version)

Eight existing entries received updates:
- "AirPods"
- "iPad"
- "M-series chips"
- "solid-state drive"
- "third party/third-party"

These revisions reflect evolving product naming conventions and technical terminology.

## Previous Update Cycles

**September 2024**: Added entries for "Apple Account," "child's Apple Account," and "Managed Apple Account"

**March 2024**: Introduced 80+ new terms related to Apple Vision Pro and accessibility features

**October 2022**: Added "Dynamic Island," "Focus filters," and system settings terminology

**Earlier versions** (2019-2021): Established foundational inclusive language guidelines and expanded accessibility terminology

## Content Structure

The guide organizes entries alphabetically (A–Z) and by category, including sections on:
- International style conventions
- Units of measurement
- Technical notation
- Inclusive writing practices
