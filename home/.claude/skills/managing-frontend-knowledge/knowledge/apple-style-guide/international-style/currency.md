# Currency

Guidelines for representing currency in Apple materials.

## Currency Symbols

- Use standard currency symbols ($, €, £, ¥, etc.)
- Place currency symbols according to regional conventions
- Include currency code when ambiguity exists (USD, EUR, GBP, JPY)

## Formatting

### Amounts
- Use appropriate decimal separators for region
- Include thousands separators based on locale
- Round appropriately for context

### Examples
- **US**: $1,234.56
- **Europe**: €1.234,56 or 1.234,56 €
- **UK**: £1,234.56
- **Japan**: ¥1,234

## International Considerations

- Specify currency when multiple currencies might be relevant
- Use ISO currency codes in technical documentation
- Consider exchange rates for pricing examples
- Localize currency formatting for regional content
