# Telephone Numbers

Guidelines for formatting telephone numbers in Apple materials.

## International Format

Use E.164 format for international numbers:
- Start with plus sign (+)
- Include country code
- Example: +1 408 555 1234 (US), +44 20 1234 5678 (UK), +81 3 1234 5678 (Japan)

## Regional Formatting

### United States
- (408) 555-1234
- 408-555-1234
- 1-408-555-1234 (with country code)

### United Kingdom
- 020 1234 5678
- (020) 1234 5678
- +44 20 1234 5678 (international)

### Japan
- 03-1234-5678
- (03) 1234-5678
- +81-3-1234-5678 (international)

## Best Practices

- Use consistent formatting within documents
- Include country code for international audiences
- Use spaces or hyphens for readability
- Avoid periods as separators in phone numbers
- Specify whether number is toll-free, mobile, or landline when relevant

## Special Numbers

- Emergency numbers: Format as displayed locally (911, 999, 110)
- Toll-free numbers: Include access information for international callers
- Short codes: Format without extra characters
