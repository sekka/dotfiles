# Navigation（ナビゲーション）

Material Design 3は、デバイスサイズに応じた適応的なナビゲーションパターンを提供します。

## ナビゲーションコンポーネントの選択

### デバイスサイズによる選択

| デバイス | 幅 | 推奨コンポーネント |
|---------|-----|-------------------|
| モバイル | <600px | Navigation Bar |
| タブレット | 600-1200px | Navigation Rail |
| デスクトップ | >1200px | Navigation Drawer |

### 目的地の数による選択

| 目的地の数 | 推奨コンポーネント |
|-----------|-------------------|
| 2個 | Tabs |
| 3-5個 | Navigation Bar |
| 3-7個 | Navigation Rail |
| 5個以上 | Navigation Drawer |

## Navigation Bar（ナビゲーションバー）

### 用途
- モバイルデバイスの主要ナビゲーション
- 画面下部に固定配置

### 特徴
- **目的地の数**: 3-5個が理想
- **配置**: 画面下部
- **アイコン**: 各目的地にアイコンとラベル
- **選択状態**: 現在地を明確に表示

### いつ使うべきか
- スマートフォンアプリ
- 頻繁に切り替える主要セクション
- フラットな情報構造

## Navigation Rail（ナビゲーションレール）

### 用途
- タブレットの主要ナビゲーション
- 画面左側に固定配置

### 特徴
- **目的地の数**: 3-7個
- **配置**: 画面左側（垂直）
- **FAB**: オプションでFABを配置可能
- **省スペース**: Navigation Drawerより省スペース

### いつ使うべきか
- タブレット、中型デバイス
- 常に表示しておきたいナビゲーション
- 画面スペースを効率的に使いたい場合

## Navigation Drawer（ナビゲーションドロワー）

### 用途
- デスクトップの包括的なナビゲーション
- 画面左側、通常は非表示

### 特徴
- **目的地の数**: 5個以上も可能
- **配置**: 画面左側（スライドイン）
- **階層**: グループ化や階層構造をサポート
- **永続的または一時的**: 常時表示またはオーバーレイ

### いつ使うべきか
- デスクトップアプリ
- 多くの目的地がある
- 階層的なナビゲーション構造

## Tabs（タブ）

### 用途
- 同じページ内のコンテンツ切り替え
- 画面上部に配置

### 特徴
- **目的地の数**: 2-6個が理想
- **配置**: 画面上部またはコンテンツ直上
- **スワイプ**: 左右スワイプで切り替え（オプション）

### いつ使うべきか
- 関連するコンテンツグループ
- 同じ階層のコンテンツ
- 素早い切り替えが必要

## ナビゲーションパターンの組み合わせ

### パターン1: Navigation Bar + Tabs
```
[Tabs: Overview | Details | Reviews]
     ↑ 画面上部

[Content Area]

[Nav Bar: Home | Search | Favorites | Profile]
     ↑ 画面下部
```
- **用途**: モバイルで、主要セクション内にさらにサブセクション

### パターン2: Navigation Drawer + Tabs
```
[☰ Drawer] [Tabs: Today | Week | Month]
     ↑         ↑ 画面上部

[Content Area]
```
- **用途**: デスクトップで、多くのセクションと詳細な分類

## ベストプラクティス

### DO（推奨）

1. **デバイスに適応**: 画面サイズに応じて適切なコンポーネント
2. **明確なラベル**: アイコンだけでなくテキストラベルも
3. **現在地の明示**: ユーザーが今どこにいるかを明確に
4. **一貫性**: アプリ全体で統一されたナビゲーション

### DON'T（避けるべき）

1. **過剰な選択肢**: 7個以上の目的地は避ける
2. **不適切な配置**: モバイルで上部にナビゲーションバーなど
3. **曖昧なアイコン**: 意味が明確でないアイコン
4. **階層の混在**: 同じナビゲーションに異なる階層を混ぜない

## アクセシビリティ

- **キーボードナビゲーション**: Tab、矢印キーで操作可能
- **スクリーンリーダー**: 各目的地が明確に読み上げられる
- **フォーカス表示**: 現在フォーカスされている項目が明確
- **タップ領域**: 48x48dp以上
