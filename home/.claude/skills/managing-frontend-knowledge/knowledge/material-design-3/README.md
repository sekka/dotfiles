# Material Design 3 Knowledge Base

Material Design 3（M3）は、Googleが開発したオープンソースのデザインシステムです。このナレッジベースでは、Material Design 3のデザイン哲学、原則、ガイドラインをまとめています。

**注意**: このナレッジベースはデザイン原則とガイドラインに焦点を当てています。コード実装の詳細は含まれていません。

## Material Design 3とは

Material Design 3は、美しく使いやすい製品を高品質で素早く構築するための適応可能なシステムです。オープンソースコードに支えられ、チームが高品質なデジタル体験を構築するのを支援します。

### Material You

Material Design 3の中核には「Material You」という概念があります。これは以下の3つの柱で構成されています：

- **Personal（パーソナル）**: ユーザーの好みに応じてカスタマイズ可能
- **Adaptive（アダプティブ）**: あらゆるデバイスやスクリーンサイズに適応
- **Expressive（エクスプレッシブ）**: 感情的で魅力的なユーザー体験を提供

## このナレッジベースの使い方

このナレッジベースは、Material Design 3でデザインする際の判断指針として使用してください：

1. **デザイン判断時**: 「なぜこのコンポーネントを使うべきか」「いつ使うべきか」の判断材料として
2. **アクセシビリティ確認**: すべてのユーザーが使えるデザインになっているか確認
3. **ブランド化計画**: Material Design 3をどのようにカスタマイズしてブランド化するか
4. **デザインレビュー**: デザインがMaterial Design 3の原則に従っているか確認
