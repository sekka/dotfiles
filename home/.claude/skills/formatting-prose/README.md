# formatting-prose

過度にフォーマットされた Markdown 文章を、流れのある散文的な文章に変換するスキル。

## 目的

生成 AI が出力した冗長で箇条書き過多のレポートを、読みやすく簡潔な文章に整形する。

## 特徴

- **リストを段落に統合**: 箇条書きを自然な文章の流れに変換
- **過度な強調を削除**: 太字・斜体を削除し、通常のテキストに
- **セクション構造の最適化**: 過度に細分化された見出しを統合
- **接続詞の挿入**: 段落間の流れを自然に

## 保持する要素

以下の要素は削除せず保持：

- `inline code`
- コードブロック
- 見出し（`##`）
- チェックリスト
- テーブル
- 真に離散的なアイテム（ブラウザ一覧、言語一覧等）

## 使い方

### 基本的な使用例

```
以下の文章を散文的にして：

[箇条書き過多の Markdown]
```

### 変換例

**入力:**

```markdown
以下の3つの理由があります：

- 理由A
- 理由B
- 理由C
```

**出力:**

```markdown
理由A、理由B、理由Cという3つの理由がある。
```

## トリガーキーワード

- 「散文的にして」
- 「箇条書きを減らして」
- 「読みやすく整形して」
- 「Markdown を整理して」
- 「リストを段落に」
