# クリエイティブ事例リサーチスキル

日本のクリエイティブ業界における最新トレンドと注目事例を調査・収集するスキルです。

## 概要

このスキルは、以下の機能を提供します:

- **トレンド調査**: 日本のクリエイティブアワード、デザインギャラリーサイト、業界メディアから最新トレンドを収集
- **制作会社・クリエイターデータベース**: 79社/人の詳細情報を分野別に整理
- **隣接分野の提案**: 検索した分野に関連する他分野の事例を自動提案

## 調査ソース

### アワード・コンテスト

- **Good Design Award**: 日本最大級のデザイン賞（毎年10月発表）
- **JAGDA Awards**: 日本グラフィックデザイン協会主催の広告・グラフィック賞
- **日本タイポグラフィ年鑑**: タイポグラフィとエディトリアルデザインの年鑑（毎年5月発表）
- **CSS Design Awards**: Webデザインの国際アワード（日次・月次・年次）
- **Awwwards**: デジタルデザインとUXの国際アワード（日次・月次・年次）
- **One Show**: 広告・インタラクティブ・デザイン等の国際アワード（毎年5月発表）

### ギャラリー・キュレーションサイト

- **MUUUUU.ORG**: 縦長のWebデザインギャラリー（月次更新）
- **SANKOU!**: Webデザインの参考事例集（週次更新）
- **Responsive Web Design JP**: レスポンシブWebデザインのギャラリー
- **I/O 3000**: ハイクオリティなWebデザインギャラリー

### 業界メディア・データベース

- **AXIS**: デザイン専門誌（月刊）
- **Web Designing**: Webクリエイター向け専門誌（隔月刊）
- **good productions**: クリエイティブ企業データベース
- **Bau-hau**: 建築・空間デザイン情報サイト

## 使用方法

1. **ユーザーからリクエストを受ける**
   - 例: 「2025年の金融業界向けWebサイトのトレンドを教えて」

2. **調査ソースから情報収集**
   - アワードサイト、ギャラリーサイトをWebSearchまたはWebFetchで調査
   - 最新情報を優先的に収集

3. **制作会社データベースから関連会社をピックアップ**
   - カテゴリテーブルを参照し、関連する会社をリストアップ
   - 各社のURL、強み、実績を提示

4. **隣接分野の提案**
   - 例: 金融 → FinTech、保険、決済サービス等の関連分野を提案
   - 他業界の類似事例を紹介

5. **Markdownレポート生成**
   - 厳選された事例（5-10件）をまとめる
   - カテゴリ、事例名、URL、特徴、なぜ注目か、を明記
