# 市場調査とトレンド分析

## 概要

市場調査、トレンド分析、競合分析、バイラルトレンド検出を包括的に支援するスキルです。

## 実行フロー

### Step 1: 市場規模の把握

#### TAM/SAM/SOM分析

- **TAM** (Total Addressable Market): 全体市場規模
- **SAM** (Serviceable Addressable Market): 到達可能市場
- **SOM** (Serviceable Obtainable Market): 獲得可能市場

#### 成長ドライバーの分析

- 技術変化
- 人口動態
- 行動変化
- 規制変化

### Step 2: ターゲットセグメント分析

#### セグメント定義

```
├── デモグラフィック（年齢、性別、所得、職業）
├── サイコグラフィック（価値観、ライフスタイル）
├── 行動（利用パターン、購買行動）
└── ニーズ（課題、欲求、目標）
```

#### ペルソナ作成

- 基本情報
- 課題と目標
- 行動パターン
- 価値観

### Step 3: 競合分析

#### 競合分類

| 分類     | 定義                       |
| -------- | -------------------------- |
| 直接競合 | 同じ課題を同じ方法で解決   |
| 間接競合 | 同じ課題を異なる方法で解決 |
| 代替手段 | 課題を放置/別の方法で対処  |
| 新規参入 | 将来の競合となりうる存在   |

#### SWOT分析

- Strengths（強み）
- Weaknesses（弱み）
- Opportunities（機会）
- Threats（脅威）

#### 差別化機会

- 機能ギャップ
- 価格ギャップ
- セグメントギャップ
- 体験ギャップ

### Step 4: トレンド分析

#### ソーシャルリスニング

- Twitter/X: ハッシュタグ、メンション
- TikTok: 視聴回数、シェア
- Reddit: アップボート、コメント数

#### トレンド評価基準

| 基準         | 合格基準     |
| ------------ | ------------ |
| 拡散力       | 高           |
| マネタイズ   | 明確         |
| 技術的実現性 | 可能         |
| 市場規模     | 10万人以上   |
| 差別化機会   | あり         |

### Step 5: ユーザー行動分析

#### 世代別の特徴

- Z世代: 短尺動画、即時性
- ミレニアル: 利便性、レビュー重視
- X世代: 実用性、信頼性

#### バイラル係数

```
K-factor = i × c
K > 1: 指数的成長
```

### Step 6: App Store Intelligence

- トップチャート分析
- レビュー分析
- キーワード分析

## 出力成果物

1. **市場規模レポート**: TAM/SAM/SOM
2. **ペルソナ文書**: ターゲットユーザー詳細
3. **競合分析マトリクス**: 比較表
4. **SWOT分析**: 4象限図
5. **トレンドレポート**: バイラルトレンド評価

## ベストプラクティス

1. **データに基づく分析**: 複数ソースで検証
2. **実用的なインサイト**: アクションにつながる推奨
3. **継続的なモニタリング**: トレンドの早期検出
4. **バイアスの認識**: 反証も探す
5. **スピード重視**: 完璧を待たない
