# マーケティングコンテンツ作成

## 概要

マーケティングコンテンツ作成、ブログ記事、LP、広告コピー、メールキャンペーンを包括的に支援するスキルです。

## 実行フロー

### Step 1: コンテンツ戦略の策定

#### コンテンツピラー

```
ピラー1: 教育的コンテンツ（ハウツー、チュートリアル）
ピラー2: ソートリーダーシップ（業界トレンド分析）
ピラー3: 製品関連（機能紹介、成功事例）
ピラー4: エンゲージメント（顧客ストーリー、Q&A）
```

### Step 2: ブログ記事作成

#### SEO最適化

```
プライマリキーワード: タイトル、H1に含む
セカンダリキーワード: H2-H3に含む
本文: 自然に散りばめる（密度2-3%）
メタディスクリプション: 160文字以内
```

### Step 3: ランディングページコピー

- ヒーローセクション: 価値提案、CTA
- 課題提起セクション: ターゲットの痛点
- ソリューションセクション: ベネフィット中心
- ソーシャルプルーフ: 顧客の声、実績数値
- FAQ
- 最終CTA

### Step 4: 広告コピー作成

- Google広告: 見出し30文字×3、説明文90文字×2
- Facebook/Instagram広告: プライマリテキスト125文字
- LinkedIn広告: B2B向けフォーマル

### Step 5: メールコンテンツ

#### オンボーディングシーケンス

```
Day 1: ウェルカムメール
Day 3: 主要機能の深掘り
Day 7: 成功事例の共有
Day 14: フィードバック依頼
```

### Step 6: コピーライティングフレームワーク

#### AIDA Framework

```
Attention（注意）: 魅力的な見出し
Interest（興味）: 問題提起と共感
Desire（欲求）: ベネフィットを提示
Action（行動）: 明確なCTA
```

#### PAS Framework

```
Problem（問題）: 読者の痛点を明確化
Agitation（煽動）: 放置した場合の悪影響
Solution（解決策）: 製品を解決策として提示
```

#### 4U Headline Formula

```
Urgent（緊急性）: "今すぐ", "期間限定"
Unique（独自性）: "業界初", "唯一の"
Useful（有用性）: "時間を50%削減"
Ultra-specific（超具体的）: "3ステップで"
```

### Step 7: コンテンツ再利用戦略

```
ピラー記事1本（3,000語）
├─ ソーシャル投稿 × 10本
├─ メールコンテンツ × 3通
├─ 動画スクリプト × 2本
└─ スライド × 1本
```

## 出力成果物

1. **ブログ記事**: SEO最適化された長文（1,500-3,000語）
2. **LPコピー**: セクション別の完成コピー
3. **広告コピー**: プラットフォーム別に最適化
4. **メールコンテンツ**: 件名/本文/CTA
5. **動画スクリプト**: YouTube、TikTok用
6. **A/Bテスト案**: バリエーション

## ベストプラクティス

1. **価値優先**: プロモーションより先に価値提供
2. **明確なCTA**: 1コンテンツ1CTA
3. **データと証拠**: 具体的な数字と事例
4. **走査しやすさ**: 小見出し、箇条書き、短い段落
5. **ブランドボイスの一貫性**: 統一されたトーン
