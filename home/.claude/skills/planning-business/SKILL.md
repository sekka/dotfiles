# ビジネス戦略とマネタイズ設計

## 概要

ビジネスモデル分析、価格戦略、財務計画、収益最適化を包括的に支援するスキルです。

## 実行フロー

### Step 1: ビジネスモデル分析

#### ビジネスモデルキャンバス（9つの構成要素）

- 価値提案: 顧客に提供する独自の価値
- 顧客セグメント: ターゲットとなる顧客群
- チャネル: 顧客へのリーチ方法
- 顧客関係: 獲得・維持・拡大の方法
- 収益の流れ: マネタイズの仕組み
- 主要リソース: 必要な資産・能力
- 主要活動: 価値を生む活動
- パートナー: 外部協力者
- コスト構造: 主要なコスト項目

#### 収益モデル類型

| モデル             | 特徴                     | 適用例           |
| ------------------ | ------------------------ | ---------------- |
| サブスクリプション | 定期課金、予測可能な収益 | SaaS、メディア   |
| フリーミアム       | 無料で獲得、有料で収益化 | ツール、ゲーム   |
| 従量課金           | 使用量に応じた課金       | API、クラウド    |
| マーケットプレイス | 取引手数料               | EC、マッチング   |

### Step 2: 価格戦略の設計

#### 価格設定アプローチ

- **コストベース**: コスト + マージン
- **バリューベース**: 顧客が感じる価値に基づく
- **競合ベース**: 競合価格を基準に設定

#### Good-Better-Best構造

```
Basic        Pro          Enterprise
- 必須機能   - 人気機能   - 全機能
- 個人向け   - チーム向け  - 組織向け
```

### Step 3: 財務計画の作成

#### 主要指標（Key Metrics）

**成長指標:**

- MRR/ARR（経常収益）
- MoM/YoY成長率

**ユニットエコノミクス:**

- CAC（顧客獲得コスト）
- LTV（顧客生涯価値）
- LTV/CAC比率（目安: 3x以上）
- Payback Period（回収期間: 目安12ヶ月以内）

**収益性指標:**

- 粗利率（SaaSは70%+）
- 営業利益率

### Step 4: ユニットエコノミクスの最適化

#### CAC削減施策

- チャネル別CAC分析
- オーガニック獲得の強化
- 紹介プログラムの実装

#### LTV向上施策

- チャーン率の低減
- アップセル/クロスセル
- カスタマーサクセスの強化

### Step 5: 実行計画の策定

#### RICE優先順位付け

```
RICE Score = (Reach × Impact × Confidence) / Effort
```

## 出力成果物

1. **ビジネスモデルキャンバス**: 9要素の分析
2. **価格プラン設計**: Good-Better-Best構成
3. **財務計画**: 12ヶ月の予測
4. **シナリオ分析**: ベース/楽観/悲観
5. **最適化施策リスト**: CAC削減、LTV向上

## ベストプラクティス

1. **データに基づく意思決定**: A/Bテストで検証
2. **段階的な実行**: 小さく始めて検証
3. **顧客中心の価値提供**: 価格ではなく価値で競争
4. **透明性の確保**: 隠れコストを避ける
5. **継続的な最適化**: 四半期ごとに見直し
