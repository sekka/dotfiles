# Names and Unit Symbols for Units of Measure

Guidelines for using unit names and symbols in Apple materials.

## General Principles

- Use standard abbreviations and symbols
- Maintain consistency in formatting
- Follow international conventions where applicable

## Common Units

### Distance and Length
- **meter (m)**: Base SI unit for length
- **centimeter (cm)**: 1/100 of a meter
- **millimeter (mm)**: 1/1000 of a meter
- **kilometer (km)**: 1,000 meters

### Weight and Mass
- **gram (g)**: Base unit for mass
- **kilogram (kg)**: 1,000 grams

### Time
- **second (s)**: Base unit for time
- **minute (min)**: 60 seconds
- **hour (h)**: 60 minutes

### Digital Storage
- **byte (B)**: Basic unit of digital information
- **kilobyte (KB)**: 1,000 bytes (decimal) or 1,024 bytes (binary)
- **megabyte (MB)**: 1,000,000 bytes (decimal)
- **gigabyte (GB)**: 1,000,000,000 bytes (decimal)
- **terabyte (TB)**: 1,000,000,000,000 bytes (decimal)

## Formatting Rules

- Include space between number and unit symbol
- Use lowercase for unit names in running text
- Capitalize unit symbols derived from proper names (W for Watt, Hz for Hertz)
