# Countries

Guidelines for referencing countries in Apple materials.

## General Principles

- Use official country names
- Respect regional sensitivities
- Follow UN conventions when appropriate
- Consider local preferences for country names

## Formatting

- Capitalize country names
- Use consistent naming across documents
- Include region identifiers when necessary for clarity
- Avoid political implications in country references

## Best Practices

- Be aware of changing geopolitical situations
- Use neutral language
- Consider using region names when broader scope is appropriate
- Verify country names with local teams for localized content
