# Web制作における参考サイトの探し方（6つの観点）

> 出典: https://quoitworks.com/blog/archives/603
> 追加日: 2026-04-13

muuuuu.org 運営者による、Web 制作での参考サイト探しの方法論。
『アイデアのつくり方』（ジェームス・W・ヤング）の5段階プロセスを踏まえた実践ガイド。

## 参考サイトを探す目的

1. クライアントとの「完成イメージ」の認識共有
2. 業界の水準感の把握
3. Web 表現エレメントの習得

## 6つの独立した観点

参考サイト探しは「一度に全部」ではなく、**6つの観点を独立させて**行う。

| 観点 | 探す対象 |
|------|---------|
| **① ターゲット層のトーン** | ターゲット層の好み・雰囲気・トーンが合うサイト |
| **② 情報設計・画面設計** | ナビゲーション構造・コンテンツ配置が参考になるサイト |
| **③ 業界マナー・コンテンツ** | 同業界のサイトに共通するマナー・必須コンテンツ |
| **④ 写真の方向性** | 使用する写真・ビジュアルのテイスト参考 |
| **⑤ インタラクション表現** | アニメーション・マイクロインタラクションの参考 |
| **⑥ 社会的認知度の高いサイト** | 多くの人が知っているサイト（信頼感の基準点） |

## アイデア生成の5段階（参考サイト探しへの応用）

1. **資料収集** — 参考サイトを大量に集める
2. **咀嚼** — よく観察し、特徴を分析する
3. **忘却** — 一旦意識から離す
4. **発想** — 組み合わせのアイデアが生まれる
5. **形成** — 具体的なデザインへと落とし込む

## 初心者が陥りやすい誤り（抜粋）

- 「自分の好み」だけで参考サイトを選ぶ → ターゲット不一致になる
- 1つの観点だけで参考サイトを集める → 視野が狭くなる
- 参考サイト集めを「模倣のため」と捉える → 学習効果が低下する

## よく使われる参考サイト集めのリソース

- **Awwwards** — 受賞サイト、インタラクション・デザイン性
- **MUUUUU.ORG** — 日本語サイト特化のギャラリー
- **SANKOU!** — 国内 Web デザインギャラリー
