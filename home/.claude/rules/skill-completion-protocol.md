# スキル完了ステータスプロトコル

全スキル（スラッシュコマンド）の出力末尾に統一ステータスを付与する。

## ステータス定義

| ステータス | 意味 |
|---|---|
| `DONE` | 正常完了。懸念事項なし |
| `DONE_WITH_CONCERNS` | 完了したが注意すべき点あり |
| `BLOCKED` | 続行不可能な障害が発生 |
| `NEEDS_CONTEXT` | 判断に必要な情報が不足 |

## 出力フォーマット

スキル出力の末尾に以下を追加する：

```
## Status: DONE
```

`DONE` 以外の場合は理由を付記する：

```
## Status: DONE_WITH_CONCERNS
- 懸念点1
- 懸念点2
```

```
## Status: BLOCKED
{続行できない理由}
```

```
## Status: NEEDS_CONTEXT
{必要な情報・確認事項}
```

## 適用範囲

`~/.claude/skills/` 配下の全スキルファイルに適用する。
