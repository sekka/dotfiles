# プランレビュー自動提案

## ExitPlanMode 後の自動提案

plan mode（ExitPlanMode）でプランが承認された後、実装開始前に以下を提案する:

「このプランを並列AIレビューしますか？ `/review-parallel <plan-file>` で複数AIの視点からレビューできます。」

- ユーザーが承認 → parallel-reviewer をプランレビューモードで起動
- ユーザーが辞退 → そのまま実装に進む

## 提案のタイミング

ExitPlanMode ツールが呼ばれ、ユーザーにプランが提示された直後に提案する。

## 提案文例

```
このプランを並列AIレビューしますか？
`/review-parallel <plan-file>` で複数AIの視点からレビューできます。

- Codex: ロジック・実現可能性の検証
- Gemini: アーキテクチャ整合性・システム影響の評価
- Copilot: 実装実用性・工数の確認
- CodeRabbit: セキュリティリスクの評価

プランファイルのパスを確認して実行してください。
```

## キーワードによるプランレビュー自動起動

以下のキーワードを含む要求を受けた場合も、プランレビューモードで起動する:

- "プランをレビュー"
- "設計をレビュー"
- "計画をレビュー"
- "plan review"
- "review plan"
- "review this plan"
