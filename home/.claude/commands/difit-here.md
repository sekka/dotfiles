# difit-here

カレントリポジトリのGit差分をdifitで表示する。

- **デフォルトブランチ上**: working tree の差分
- **フィーチャーブランチ上**: merge-base から現在のブランチまでの差分
- **cmux 環境**: ブラウザペインで表示（`CMUX_WORKSPACE_ID` が設定されている場合）
- **その他の環境**: システムブラウザで表示

## いつ実行するか

ユーザーが明示的に呼んだ場合に加え、以下の状況でも能動的に実行してよい：

- 実装・修正が一段落し、差分を確認したい自然なタイミング（コミット前、PR作成前など）
- ユーザーが「差分を見たい」「変更を確認したい」と言ったとき
- 大きな変更をレビューする前

## 実行手順

1. `~/dotfiles/scripts/git/difit-here.sh` をバックグラウンドで実行する（`run_in_background: true`）
2. ユーザーに「差分ビューアを開きました」と伝える
3. ブラウザを閉じると自動でサーバーが終了する旨を伝える
