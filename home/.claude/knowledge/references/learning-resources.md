## Harvard CS50 日本語版

> Source: https://togetter.com/li/2029531

ハーバード大学の CS 入門講座「CS50」の日本語版が無料公開されている。

**URL**: https://cs50.jp  
**対象**: プログラミング初心者〜中級者  
**用途**: CS 基礎の体系的な習得、チームへの学習リソース共有

### 学習内容
- C言語・Python の基礎
- アルゴリズム・データ構造・メモリ管理
- Web 開発（HTML / CSS / JavaScript / Flask）
- データベース（SQL）
- サイバーセキュリティ入門

### 特徴
- 動画は日本語字幕付き、講義ノートも日本語訳あり
- 英語が聞き取りやすく英語学習と並行可能

### ChatGPT で無料英語学習（4スキル対応）

> Source: https://cancam.jp/archives/1548907  
> Added: 2026-04-13

英国留学中のライターが実践するChatGPT活用英語学習法。スピーキング・ライティング・リーディングの4スキルを無料でカバー。

**主な方法:**
- スピーキング: カフェ注文・ホテルチェックインなどロールプレイ→修正
- ライティング: 日記添削、TOEIC/IELTS基準のエッセイフィードバック
- リーディング: レベル指定の読解問題生成、要約比較
- ボーナス: フォーマル文→SNS口語表現変換

### 孫正義元秘書おすすめ 無料英語学習ツール4選

> Source: https://www.huffingtonpost.jp/entry/story_jp_673d9879e4b0060e54da6b69  
> Added: 2026-04-13

1年で英語をマスターした三木雄信氏（ソフトバンク元社長秘書）が推奨するツール。

**ツール一覧:**
- **Language Reactor** — Netflix等で字幕対訳学習
- **Mikan** — 単語アプリ
- **ELSA Speak** — AI発音矯正
- **endless（旧：AI無限トーク）** — AIとの英会話練習

**習慣化のコツ**: 食後など「トリガー」を作る／ながら学習・スキマ学習の積み重ね

## スケジュール作成フロー（WBS/PMBOK）

> Source: https://x.com/dx_saru/status/1951113033809535233 (@dx_saru)  
> Added: 2026-04-13

プロジェクトマネジメントにおけるスケジュール作成の基本フロー。

```
タスク洗い出し → 優先順位付け → スコープ確定 → WBS作成
→ タスク階層分け・粒度統一 → 作業時間見積もり
→ 依存関係整理・作業順序確定 → スケジュール完成
```

**出典**: PMBOK のスコープ計画・スケジュール計画に準拠
