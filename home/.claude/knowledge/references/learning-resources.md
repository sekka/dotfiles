## Harvard CS50 日本語版

> Source: https://togetter.com/li/2029531

ハーバード大学の CS 入門講座「CS50」の日本語版が無料公開されている。

**URL**: https://cs50.jp  
**対象**: プログラミング初心者〜中級者  
**用途**: CS 基礎の体系的な習得、チームへの学習リソース共有

### 学習内容
- C言語・Python の基礎
- アルゴリズム・データ構造・メモリ管理
- Web 開発（HTML / CSS / JavaScript / Flask）
- データベース（SQL）
- サイバーセキュリティ入門

### 特徴
- 動画は日本語字幕付き、講義ノートも日本語訳あり
- 英語が聞き取りやすく英語学習と並行可能
