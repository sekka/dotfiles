# 調査: playcanvas/editor - WebGL/WebGPU/WebXR ビジュアルエディタ

## 調査対象

https://github.com/playcanvas/editor

## 概要

**PlayCanvas Editor** は WebGL/WebGPU/WebXR アプリを構築するためのビジュアル編集環境です。https://playcanvas.com でアクセスできます。ゲームやインタラクティブな 3D コンテンツをブラウザ上で作成できます。

## 効用・ユースケース

- ブラウザベースの 3D ゲーム・インタラクティブアプリの開発
- WebGL/WebGPU を使ったリアルタイムレンダリング
- WebXR による VR/AR コンテンツ作成
- チームでの 3D プロジェクト共同作業

## メリット

- **ビジュアルエディタ**: コードなしでシーンを構築できる
- **ブラウザベース**: インストール不要、どこからでもアクセス可能
- **WebGPU 対応**: 最新のグラフィックス API に対応
- **オープンソース**: エディタのフロントエンドがオープンソース（MIT ライセンス）
- **活発なコミュニティ**: Discord、Reddit、フォーラムで活発なサポート
- **ローカル開発**: Node.js 18+ でローカル開発サーバーも構築可能

## デメリット・注意点

- **ゲーム開発専用**: 汎用ツールではなく 3D コンテンツ/ゲーム開発に特化
- **学習コスト**: 3D エンジンの概念（シーングラフ、物理エンジン等）を学ぶ必要あり
- **dotfiles との関連性**: dotfiles の日常開発環境設定とは直接関係が薄い

## 判断のポイント

3D ゲームや WebGL コンテンツの開発に興味がある場合は有用。dotfiles の日常開発環境構築とは用途が異なるため、個人プロジェクトとして検討するとよい。
